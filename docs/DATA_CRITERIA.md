# データの選定基準

## データの出所（品名・醸造所リスト）

- **醸造所名（品名）**: [全国地ビール醸造者協議会（JBA）会員リスト](https://beer.gr.jp/member.html) の銘柄名・会員名を土台に、手動で `scripts/build_japan_breweries.cjs` に登録しています。
- **特徴**: JBAの会員リストページには「特徴」の欄はありません（銘柄名・住所・電話・レストラン有無のみ）。そのため、詳細画面に表示する「このビールの特徴」は、同じスクリプト内で**こちらで追記した分だけ**表示しています。特徴を増やしたい場合は `build_japan_breweries.cjs` の各醸造所の末尾に 9 番目の要素として文字列を追加し、`node scripts/build_japan_breweries.cjs` で JSON を再生成してください。

---

## いまのアプリの記録単位

- **醸造所（生産地）単位**で「飲んだ」を記録している。
- 地図のピン = 醸造所の場所。チェック = 「この醸造所のビールを飲んだ」という意味。

---

## いまのリストの選定基準（醸造所）

1. **全国網羅**: 47都道府県すべてに最低1件以上、醸造所を配置。

2. **有名どころ**: 認知度の高い醸造所を優先して掲載。
   - 例: 常陸野ネスト、COEDO、箕面ビール、ベアード、ヨ・ホー、大沼ビール、ニセコビール、羊蹄山麓、忽布古丹、ベアレン（盛岡）、TAKAO BEER、UCHU BREWING、富士メイスビア など。

3. **盛んな地域は多め**: 地ビール・クラフトビールの醸造所が多い地域は件数を多めに登録。
   - **北海道**: 札幌・函館・小樽・旭川・網走・大沼・ニセコ・倶知安・上富良野 など（13件前後）。
   - **東京・神奈川**: 都内ブルワリー・横浜・湘南・鎌倉・箱根 など（18件前後）。
   - **長野・山梨**: 安曇野・松本・軽井沢・志賀高原・八ヶ岳・小淵沢 など（12件前後）。
   - **静岡**: 御殿場・沼津・浜松・伊豆・富士宮 など（8件前後）。
   - **京都・大阪・兵庫**: 京都市内・箕面・道頓堀・神戸・堺 など（20件前後）。
   - **福岡**: 福岡市・北九州・久留米 など（6件前後）。
   - **沖縄**: 那覇・名護・石垣 など（4件前後）。

4. **ベース**: [全国地ビール醸造者協議会（JBA）会員リスト](https://beer.gr.jp/member.html) を土台に、上記の方針で追加・調整。会員のいない都道府県は「〇〇県地ビール」など代表1件でカバー。

5. **結果**: 約150件以上。  
   - 基準は「**全国網羅**」「**有名どころ**」「**盛んな町は多く**」。

## 「アサヒ飲んだ」「ヱビス飲んだ」のように銘柄で記録したい場合

- 記録単位を **醸造所 → 銘柄（ビールの名前）** に変えるイメージ。
- 全国には銘柄が非常に多いので、**どの銘柄をリストに載せるか**の基準が必要。

### 銘柄リストの選定案

| 案 | 内容 | 長所 | 短所 |
|----|------|------|------|
| **A. 醸造所の代表銘柄** | 今の133醸造所について「主な銘柄」を1〜数本ずつ登録する（例: 小樽麦酒、常陸野ネスト、箕面ビール） | すでにある醸造所データと対応しやすい | 1醸造所で複数銘柄ある場合の線引きが必要 |
| **B. 市販されている地ビール** | スーパー・酒屋・通販でよく見かける「地ビール」銘柄に絞る | 実際に「飲んだ」と記録しやすい銘柄が並ぶ | リストの作成・更新が手間。定義が曖昧 |
| **C. JBA会員のブランド名** | 協議会会員の「ブランド名」「商品名」をリスト化する | 公的な情報に近い | 協議会サイトに銘柄一覧がなく、別途調査が必要 |
| **D. ユーザー登録中心** | 最初は少数の代表銘柄だけ用意し、あとは「飲んだビールを追加」でユーザーが銘柄を登録 | 数が膨れない。自分が飲んだものだけ増える | 地図のピンとの対応（どこの銘柄か）をどうするか要検討 |

### 運用のイメージ（銘柄で記録する場合）

- **地図**: これまで通り「醸造所の場所」にピンを立てる。
- **記録**: 「〇〇醸造所の △△ビール を飲んだ」のように、**銘柄**でチェック／メモする。
- **データ**: 醸造所 ID に紐づく「銘柄リスト」（例: 小樽麦酒、網走ビール など）を持たせる。

---

## まとめ

- **今の基準**: リストに載せているのは **「全国地ビール醸造者協議会の会員である醸造所」＋ 都道府県を埋めるための補足1件**。記録は **醸造所単位**（「この醸造所のビールを飲んだ」）。
- **銘柄で記録する場合**: 上記 A〜D のような選定基準で「載せる銘柄」を決める必要がある。運用イメージは「醸造所＝ピン」「銘柄＝飲んだ記録」にすると、今の地図コンセプトを活かしやすい。
